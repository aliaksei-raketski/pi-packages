# Pi Packages

Monorepo for public npm-distributed [Pi](https://pi.dev) packages.

## Packages

| Workspace | npm package | Install |
| --- | --- | --- |
| `angular-developer` | `@aliaksei-raketski/pi-angular-developer` | `pi install npm:@aliaksei-raketski/pi-angular-developer` |
| `taiga-ui-docs` | `@aliaksei-raketski/pi-taiga-ui-docs` | `pi install npm:@aliaksei-raketski/pi-taiga-ui-docs` |

## Development

```bash
npm run pack:dry-run
npm run sync
```

## Publishing

Packages are published to the public npm registry from GitHub Actions.
